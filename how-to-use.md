# How to Use SkinRave

Using SkinRave is easy, whether you are a new user or experienced in CS2 case openings.

## Steps:
1. Sign up using Steam at [SkinRave.gg](https://skinrave.gg)
2. Redeem the promo code **GITHUB** for free case rewards
3. Choose a case and open it
4. View your items and upgrade or trade as you like
5. Verify each roll using our provably fair system

SkinRave provides a clean interface and smooth animations for an enjoyable experience.
